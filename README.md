
HBnB is a simplified accommodation platform inspired by AirBnB.
The main objective of this project is to practice software architecture, object-oriented design, and structured backend development.

The project is divided into several stages, starting with system modeling and gradually moving toward full implementation and data persistence.
It highlights the importance of clean architecture, clear separation of responsibilities, and scalable code organization.

Part 1 – UML Design

Part 1 is dedicated to designing the system before writing the actual code.

During this phase, we created:

A High-Level Package Diagram to represent the overall layered architecture.

A Detailed Class Diagram to define the main business entities and their relationships.

Sequence Diagrams to illustrate how different API calls flow through the system layers.

The goal of this phase is to build a solid architectural foundation that ensures consistency, clarity, and maintainability for the future implementation of the HBnB application.README.md
